# CRUD Básico de Livros em Java

Este é um projeto simples que implementa um CRUD (Create, Read, Update, Delete) básico para operações de adicionar, remover e editar livros em um banco de dados MySQL, utilizando Java.

## Tecnologias Utilizadas

- MySQL: Banco de dados relacional utilizado para armazenar informações sobre os livros.
- Java: Linguagem de programação utilizada para criar o backend da aplicação.
- Spring Boot: Framework utilizado para criar a aplicação web.
- Thymeleaf: Utilizado para criar a interface de usuário básica.

## Funcionalidades

- Adicionar livros ao banco de dados, incluindo informações como título, autor, ano de publicação, etc.
- Visualizar todos os livros cadastrados.
- Editar informações de livros existentes.
- Remover livros do banco de dados.
